Lesson one: Basics of HTML

HTML stands for Hypertext Markup Language. It is used for web development. It is considered as one of the best coding languages for beginners before Python even after being so old. Because HTML is very easy to learn. You can learn the basics in just 10 minutes cuz of how simple it is. Without further ado, let's get started!

a) Every HTML code starts like this:


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Awesome Page</title>
</head>
<body>
  <h1>Hello, world!</h1>
  <p>This is my first web page.</p>
</body>
</html>
  • <!DOCTYPE html> → Tells the browser that it’s HTML5
  • <html> → wraps the whole page
  • <head> → stuff for the website like title, fonts etc.
  • <body> → all the visible content of the website goes here

b) Common tags of HTML

<h1> → <h6> → Headings (bigger to smaller)

<a> → links and many more.

<button> → creates a button (i.e. - <button>Click me</button>)
